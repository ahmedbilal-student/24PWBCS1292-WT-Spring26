Web Technologies Lab 05 & 06
CSS Layout Techniques & Responsive Design

This repository contains the implementation of Lab 05 & Lab 06 for the Web Technologies (CS-224L) course.
The lab focuses on modern CSS layout techniques and responsive web design.

Topics Covered:

The following CSS concepts are implemented in this lab:

CSS Display Properties

Float-Based Layouts

CSS Positioning

Flexbox Layout System

CSS Grid Layout System

Media Queries & Responsive Design

Mobile-First Design Approach

Learning Objectives:

After completing these tasks, students will be able to:

Understand different CSS display properties

Create layouts using floats

Use CSS positioning effectively

Build responsive layouts using Flexbox

Design complex page structures using CSS Grid

Implement media queries for responsive websites

Combine multiple layout techniques in modern web design

Technologies Used:

HTML5

CSS3

Flexbox

CSS Grid

Media Queries

Responsive Design Principles

Responsive Breakpoints:

This project follows the Mobile-First Approach:

Mobile: 0px – 767px
Tablet: 768px and above
Small Desktop: 1024px and above
Large Desktop: 1200px and above
Extra Large Screens: 1400px and above

Features:

✔ Responsive layouts
✔ Modern CSS techniques
✔ Mobile-first design
✔ Flexible navigation systems
✔ Grid and Flexbox based layouts
