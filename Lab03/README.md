# CSS Lab 03 & 04

**Course:** CS 224L – Web Technologies
**Topic:** CSS Fundamentals – Styling and Visual Design

---

## 📌 Overview

This project implements the exercises from **Lab 03 & 04** focusing on CSS fundamentals.
The goal is to practice styling HTML pages using **selectors, typography, backgrounds, images, and visual effects** with external CSS files.

---

## 🧪 Exercises Implemented

### 1. CSS Selectors

Practice with:

* Element selectors
* Class selectors
* ID selectors
* Combinators
* Pseudo-classes and pseudo-elements

### 2. Typography Showcase

Demonstrates:

* Google Fonts
* Different font sizes and styles
* Text alignment and transformation
* Text shadow effects

### 3. Background Styling

Examples include:

* Solid color backgrounds
* Background images
* CSS gradients

### 4. Image Styling

Gallery demonstrating:

* Rounded and circular images
* Image borders and shadows
* Hover effects and transitions
